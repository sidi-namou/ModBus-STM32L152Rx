
In Src/main.c you will find an implementation of modbus library

The library is located in Drivers/MBMESN there you will find:
MBMESN_errorstatus.h contains all types of error of modbus protocol,
MBMESN_hardware.h contains hardware layer of modbus protocol,
MBMESN_modbusrtu.h contains link layer of modbus protocol,
MBMESN_slavepdu.h  contains application layer of modbus protocol.
